Faculty Directory for Multi Universities

This folder contains two sub folders

  • Build Code
  • Source Code

Build Code

  • This folder contains all files of this project generated by React Js (framework of Javascript) which is non readable
  • Just open index.html by going into this folder or click here to start
  • To see actual coding files head towards Source Code folder

Source Code

  • This folder contains actual coding files written in React JS with pure JavaScript framework.
  • BackendEnd is built in attached with Google Firebase Authentication.
  • To run this, you have to download and install NPM Package Manager with Node Js
    • Then Open Console and run npm install
    • After it will install all dependencies: hit npm start
